# Blink hello world for CJMCU Beetle

## Install

As a dependency, you have to have PlatformIO installed. Please see [PlatformIO installation] documentation.

```
$ platformio run --target upload
$ platformio device monitor
```

## Parts List

* [CJMCU Beetle]
